# AURA 🌙

やさしい運勢ガイダンス Webアプリ

## 概要
AURA は、月と日のリズム、
今の心の状態、そしてその日のサインを重ねて、
「今日をどう過ごすか」のヒントを提示する
やさしい運勢ガイダンスアプリです。

未来を断言する占いではなく、
無理をしないための“指針”を届けます。

## このアプリで見ているもの
- 月と日のリズム
- 今の心の状態（質問形式）
- その日に現れる小さなサイン

これらを組み合わせて、
今日の流れを読み取ります。

## 主な機能
- 今日の運勢スコア（0〜100）
- 今日の行動指針（動く／整える／休む）
- やさしいメッセージ表示
- スマホ対応 Webアプリ

## 技術構成
- HTML
- CSS
- JavaScript
- サーバーレス（静的）

## コンセプト
占いで決めつけない。
不安を煽らない。
「今日はこれでいい」と思えるためのアプリ。
